# Code of Conduct

## Our Commitment

We are committed to creating a respectful, inclusive, and harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and discuss ideas, not people.
- Provide actionable feedback.
- Accept responsibility and learn from mistakes.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Deliberate disruption of discussions or project work.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct. They may remove, edit, or reject comments, commits, issues, or contributions that violate these standards.

## Scope

This code of conduct applies in all project spaces, including issues, pull requests, and community discussions.

## Reporting

If you experience or witness unacceptable behavior, open a private security report as described in `SECURITY.md`.
